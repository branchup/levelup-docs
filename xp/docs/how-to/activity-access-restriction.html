Unlock an activity at a certain level

info

This feature requires the availability plugin Level Up XP Availability to be installed and enabled. The Moodle setting enableavailability must also be enabled.

  1. Navigate to the settings of the activity or resource to restrict access to
  2. Locate the section Restrict access
  3. Click the button Add a restriction and select Level
  4. Choose the desired level from the dropdown and save the changes

tip

This can also be applied to whole sections.

More on access restriction at Moodle documentation.
