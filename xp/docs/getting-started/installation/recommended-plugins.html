Recommended plugins

To fully benefit from all the features offered by XP and XP+, some additional plugins may need to be installed. All of these are free and available directly from moodle.org.

Availability XP​

The plugin enables instructors to conditionally allow or restrict access to activities or sections in courses based on a learner's level. View on moodle.org

Enrol XP​

With this plugin, learners can automatically be enrolled in a course based on the level they attained in another course. View on moodle.org

Shortcodes​

Shortcodes are code snippets that instructors can use to enhance their course content. View on moodle.org
