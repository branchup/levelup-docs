# The block cannot be added

If you cannot see the Level Up XP block when trying to add a block work through the checks below.

## Check where you are trying to add the block

XP can run either **per course** or **sitewide**. If you are trying to add the block to the **Dashboard** or **Site home** while XP is configured **per course**, it is expected that the Level Up block does not appear in the "Add a block" list for those pages. In that case, either:

- Add the block inside individual courses, or
- Ask an administrator to switch XP to [sitewide mode](../../getting-started/per-course-or-site.mdx).

## Check that the block is enabled and not locked

Even if the plugin is installed, the block itself can be **disabled** or **locked** by an administrator in Moodle's block management page. If it is disabled or locked, it will not appear in the "Add a block" list.

1. Navigate to _Site administration > Plugins > Blocks > Manage blocks_.
2. Find the **Level Up XP** block in the list.
3. Confirm that:
   - The block is **enabled**.
   - The block is **unlocked** (if it is locked, unlock it).

After enabling and unlocking the block, return to the page where you want to add it and try again.
