# Controller for route not found

This error occurs when _Level Up XP_ cannot identify what page to display. This is likely to be a due to a web server configuration issue related to the Moodle setting `slasharguments`.

Please read either of the following for more details:

- [IIS considerations](/installation/iis-considerations.mdx)
- [Nginx considerations](/installation/nginx-considerations.mdx)

Alternatively, refer to the [Moodle documentation on slash arguments](https://docs.moodle.org/en/Using_slash_arguments).
