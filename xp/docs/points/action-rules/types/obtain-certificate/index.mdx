The **Obtain a certificate** action awards points to learners when they receive a certificate issued by the [Custom certificate](https://moodle.org/plugins/mod_customcert) plugin. This is a popular third-party plugin that lets educators design and issue certificates directly within Moodle.

Pairing certificates with points creates a natural reward moment: learners complete their work, earn their certificate, and are immediately recognised with points. It reinforces the value of the certificate and gives learners an extra reason to see their work through to the end.

## Requirements

- The [Custom certificate](https://moodle.org/plugins/mod_customcert) plugin version 5 or later must be installed, which itself requires Moodle 5.0 or later.

## Setting it up

1. Navigate to the **Action rules** page.
2. Add a new action and select **Obtain a certificate**.
3. Choose the **Specific activity** condition and select your certificate activity, such as "Program certificate".
4. Set the number of points to award.

From this point on, whenever a learner receives a certificate, they will earn the configured points.
