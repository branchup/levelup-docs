XP/XP+ 20.0​

April 20, 2026

New features ✨

  • Action rules: A simpler, easier, and flexible way to award points! (docs)
  • Limit how many times and how often actions give points (docs) (XP+)
  • View logs and points given by a specific action or completion rule (XP+)
  • Create rules targetting activities tagged with a specific tag (docs) (XP+)
  • Award points when a certificate is issued by the "Custom certificate" plugin (docs) (XP+)
  • Admins can set default values for the action rules

Quality of life 💪

  • The new default rules for new courses are explicit and easy to understand
  • The logs display which rule the points originated from (XP+)
  • The logs can be filtered for a specific rule (XP+)

Bug fixes 🐛

  • Improved accessibility of some dropdown menus
  • Privacy provider reports the leaderboard participation flag (XP+)

Technical changes 🛠

  • Compatibility with Moodle 5.2
  • Logs migration to a new table will take place after the upgrade
  • Default event rules could apply alongside default action rules

Additional notes 👀

  • Read the upgrade notes for technical details.
  • Read our release blog post to learn more.

XP/XP+ 18.0​

April 14, 2025

We dropped the v3. prefix of XP and the v1. prefix of XP+. XP v18 is the version following v3.17 and XP+ v18 is the version following v1.17. From this release, XP and its compatible XP+ version will have the same number.

New features ✨

  • Admins can automatically pre-populate the leaderboards with XP participants
  • Leaderboards can be configured to allow users to opt-out (XP+ Premium)
  • Administrators can enforce that all leaderboards support opt-out (XP+ Premium)

Quality of life 💪

  • Improved the column sizes of the report table
  • Improved accessibility of the levels management page
  • Improved accessibility of rule to match activities by name
  • Admins are warned before an upgrade if XP+ will be disabled
  • Admins will receive an notification if XP+ becomes disabled

Bug fixes 🐛

  • Filtering logs by group had no effect
  • Viewing the leaderboard of "All participants" was showing the user's group leaderboard
  • XP data source in custom reports was not available in Moodle pre-4.4 (XP+)
  • The drop form was not declaring the type of some fields (XP+)

Technical changes 🛠

  • The addon XP+ will be disabled when its version is older than XP
  • Great expansion of our automated tests
  • Compatibility with Moodle 5.0

Additional notes 👀

  • Read our release blog post to learn more.
