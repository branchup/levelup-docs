# Recommended plugins

To fully benefit from all the features of Quest, we recommend installing these additional plugins. All of these plugins are available for free.

## Quest Availability

The plugin enables instructors to conditionally allow or restrict access to activities or sections in courses based on a learner's progess in quests, achievements and challenges. [Get from our repository](https://github.com/branchup/moodle-availability_gearup/tags)

## Shortcodes

Shortcodes are code snippets that instructors can use to enhance their content, so of Quest features rely on it. [Get from moodle.org](https://moodle.org/plugins/filter_shortcodes)

## Level Up XP

Award points to learners for completing their quests and challenges, and create objectives based on their levels or points. Level Up XP integrates well with Level Up Quest. [Get from our website](https://levelup.plus/xp/)
