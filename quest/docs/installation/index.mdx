# Installing Level Up Quest

Level Up Quest is a Moodle plugin of type `block`. Its code name is `block_gearup` indicating that all files will be contained in the `blocks/gearup/` folder.

Please download the zip file that we provided you after the purchase, it should be named `block_gearup.zip` and contain a single folder named `gearup`. If you have lost the file, please contact support.

## Zip upload

:::warning

This method is not always available. It depends on the Moodle adminstrator and hosting provider.

:::

1. Locate the zip file that we provided you, it should be named `block_gearup.zip`.
2. Login to your Moodle site and navigate to _Site administration > Plugins > Install plugins_.
3. Upload the zip file in the form, and follow the steps.
4. Consider installing the [recommended plugins](./recommended-plugins.mdx).
5. [Add the block](./usage.mdx) and access it to enter the licence key.

For more information on installing plugins, please visit the [official Moodle documentation](https://docs.moodle.org/en/Installing_plugins).

## Manual installation

1. Locate the zip file that we provided you, it should be named `block_gearup.zip`.
2. Extract the content of the zip file in the `blocks` folder of your Moodle installation.
3. If you have extracted the content at the right place, the following file should be present: `blocks/gearup/version.php`.
4. Visit the admin notifications page to trigger the installation.
   - Navigate to _Site administration > Notifications_.
   - Or directly visit _your.moodle.example.com_**/admin/index.php**.
5. Follow the Moodle installation process.
6. Consider installing the [recommended plugins](./recommended-plugins.mdx).
7. [Add the block](./usage.mdx) and access it to enter the licence key.

For more information on installing plugins, please visit the [official Moodle documentation](https://docs.moodle.org/en/Installing_plugins).
